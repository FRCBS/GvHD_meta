# GvHD_meta
HSCT GvHD GWAS and gene expression meta-analysis

This repository contains the scripts for running gene-level summary from Plink-formatted GWAS results with MAGMA, downloading and processing of gene expression data from GEO, and rank-based meta-analysis combining these data.

## code (./src)

`main_csc_magma.sh` Shell script for calling subscripts for running MAGMA analysis

`AGVHD.R` R script for running meta-analysis and plotting for aGvHD data
